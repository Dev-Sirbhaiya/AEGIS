# SOP: Unauthorized Access at Airside Gate

## Severity: HIGH (Level 4)

## Trigger Conditions
- Access card presented at airside gate is flagged as unauthorized or suspended
- Individual attempts to bypass access control (tailgating, forced entry)
- Access granted to restricted zone without matching security clearance level
- Multiple failed access attempts at same gate within 5 minutes
- CCTV detects individual in restricted zone without visible ID/lanyard

## Immediate Actions
1. Dispatch nearest patrol officer to gate location immediately
2. Lock down the compromised gate (trigger electronic lock via SCMT)
3. Review CCTV footage of last 10 minutes at affected gate
4. Attempt radio/intercom contact with individual if still visible
5. Check access control logs for badge ID and owner identity
6. Notify Terminal Duty Manager of potential airside security breach

## Do NOT
- Do NOT confront the individual without backup if armed or aggressive
- Do NOT unlock the gate until identity is confirmed and cleared
- Do NOT assume it is a system error without physical verification
- Do NOT delay action waiting for supervisor approval on Level 4+ incidents

## Escalation Criteria
- Individual is unidentified or flees → escalate to Level 5, notify APD/SPF
- Weapons or contraband suspected → escalate to Level 5, activate STF protocol
- Multiple simultaneous breaches → escalate to Level 5, full airside lockdown
- Individual reaches runway or aircraft → Level 5, CAAS immediate notification required

## Post-Incident
- Document breach time, gate ID, badge ID (if captured), and officer response time
- Submit CAAS Occurrence Report within 4 hours if airside boundary was breached
- Review access control system for vulnerabilities
- Debrief responding officer within 24 hours
- Log in AEGIS incident database with evidence attachments
